# ⚖️ Telif Hakkı & Lisans

© 2025 **[Atakan IŞIK / MART 2026 ADAYI]** — Tüm hakları saklıdır.

Bu proje, **5846 sayılı Fikir ve Sanat Eserleri Kanunu (FSEK)** kapsamında fikri mülkiyet koruması altındadır. Kaynak kodu, tasarımı ve içeriği izinsiz kopyalanamaz, dağıtılamaz veya ticari amaçla kullanılamaz.

---

## 📋 Koruma Kapsamı

| Unsur | Açıklama |
|---|---|
| **Kaynak kodu** | HTML, CSS ve JavaScript dosyalarının tamamı |
| **Görsel tasarım** | Renk paleti, tipografi, bileşen düzeni |
| **UI mimarisi** | Panel yapısı, etkileşim ve navigasyon akışı |
| **İş mantığı** | Soru üretme, kaynak gösterme, PDF yönetim akışları |
| **Eser adı & konsepti** | "Soru Havuzu" platformunun bütünsel fikri |

> Türk hukuku uyarınca telif hakkı, eserin yaratılmasıyla birlikte **tescile gerek olmaksızın** kendiliğinden doğar.

## 🛡️ İhlal Durumunda

FSEK kapsamında aşağıdaki yollara başvurulabilir:

- **İhtarname** gönderilmesi
- **Tazminat davası** (FSEK m. 68)
- **Cezai şikayet** — hapis ve/veya adli para cezası (FSEK m. 71–72)
- Barındırma sağlayıcısına **içerik kaldırma bildirimi**
- Ticaret mahkemesinde **ihtiyati tedbir** talebi
